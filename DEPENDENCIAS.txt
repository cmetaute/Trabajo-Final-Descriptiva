  - Propósito: Cálculo de momentos estadísticos
  - Usado para: Asimetría (skewness) y curtosis (kurtosis)
  - Repositorio: CRAN
  - URL: https://cran.r-project.org/package=moments

forecast >= 8.20
  - Propósito: Análisis de series temporales
  - Usado para: Funciones auxiliares de análisis temporal
  - Repositorio: CRAN

# SOLUCIÓN DE PROBLEMAS
Si un paquete no se instala:

1. Verificar conexión a internet
2. Actualizar R a la última versión
3. Limpiar caché de paquetes: remove.packages("nombre_paquete")
4. Reinstalar: install.packages("nombre_paquete", dependencies = TRUE)
5. En Linux, instalar dependencias del sistema:
   sudo apt-get install r-base-dev libcurl4-openssl-dev libssl-dev libxml2-dev
